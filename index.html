About CodeLikePros

We are a nimble bunch -a team of software professionals from diverse backgrounds like the world's largest enterprise software firm, the hottest startups, serial entrepreneurs and professional trainers.

We write software for a living, and over the years have acquired not only the proficiency in writing quality software, but also the insider knowledge of the industry:

  • What skills will actually get you a job today?
  • What changes in your coding methodology can take you from an unrewarding job, to the best jobs out there?
  • What makes you an average developer lost in the crowd and what can make you stand out like an A player?
  • How you sell your expertise while still maintaining the upper hand in an interview?

We want to make the job market more approachable and predictable for the aspirants. We also want to help provide real value to the organisations hiring them, so that everybody wins.

What about us?
We are still working professionals, who want to pass on our insights and leverage - that's why we are starting small. You will be learning in a homely environment when you join us - not a boring training center. We have got great results so far, and are super excited about the rest of the journey!

Who is this for?

Our courses are designed to take you towards employment and personal fulfillment, no matter where you are currently at.

You can be a totally new to web/mobile development and want to seize the opportunity to get involved now.

Or you could already know the basics and want to know more to become dangerous!! You will become eligible to apply at all entry level jobs and we'll help you through the entire process.

If you are already employed, then you can make the most out of our advanced courses and unleash your full potential. There is always a better job out there. Level up!

I dont know jack about this stuff!

Start with the HTML/CSS/JS Primer. Learn to build interactive websites. End it with the course on Web Development. If you want a job, also take the specialized course on job interviews because a lot of things in there are really going to make the difference.

I know some concepts but my resume lacks any real projects.

Start by learning to build interactive websites followed by a course on Web Development. Then you can either take an advanced frontend development course or go full stack by taking a course on backend/API development. Take the course on job interviews too if that's your goal.

I am a web/mobile developer. I want to increase my salary.

Congratulations, you are already a pro!

You should take the advanced courses in either Frontend or Backend development based on your interets. They will relly give a major boost to your skills. However, don't forget to take the course on interviews as that will make a lot of difference.

If you are targeting salaries above 10LPA, also take the course on algorithms.

I want to quickly build mobile apps.

We'll make it easy for you to start building your own apps. Take the course on mobile backend development and one of either Android or Hybrid (iOS+Android) development. That's all you need, we'll take care of the rest!

I have an amazing startup idea and want to code it out!

Most ideas are more than the technology behind them. However, a Minimum Viable Product(MVP) is the best way to see if your idea resonates with the market.

In our specialized course on coding your startup, we'll teach you how to code out an MVP in the quickest possible way. This will include the front end, the backend/APIs and the Database. You can replicate this for any number of your ideas.

HTML/JavaScript/CSS Primer

HTML, JavaScript, CSS Primer

Course Description

We'll teach you the basics of the three core technologies required for building anything for the web. This is for people who have no prior experience with these technologies or never quite understood it in college.

This will enable you to build very simple web documents, but the major purpose of this course is to prepare you for the more advanced courses in Web Development.

Building Interactive Websites

Course Description

This course will also provide a smooth transition to the course on building web applications, which also interact with a database, and are much more complex in terms of functionality.

Frontend Development
Web Application Development

Web Application Development

Project Description

In this course we'll teach you how to build a modern web application. A web application is different from a website in the sense that it interacts with a backend database(DB) and generates web pages dynamically based on the data it recieves from the DB.

You'll learn the concepts of routing, DB integration, HTML templating and sessions among others. By the end of this course you'll have built a fully functional website like reddit/soundcloud.

Building APIs for Web Applications

Building APIs for Web Applications

Project Description

APIs are very important in the context of web development. They let you decople your presentation layer with the database layer and also enable third parties to utilize the capabilities of your service.

We'll teach you how to build RESTful JSON APIs, the standards which should be followed while building them and how to ensure high performance with minimal coding.

Backend Development on NodeJS

Backend Development on NodeJS

Project Description

Node.JS is suitable for IO bound applications where the server has to wait between requests. Node.JS with it's async event model makes it very easy to build realtime web applications like multiplayer games and stock trading applications.

In this course we;ll teach you basics of NodeJS and how to build a web app using Sails, a NodeJS framweork.
