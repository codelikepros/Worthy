About CodeLikePros

We are a nimble bunch -a team of software professionals from diverse backgrounds like the world's largest enterprise software firm, the hottest startups, serial entrepreneurs and professional trainers.

We write software for a living, and over the years have acquired not only the proficiency in writing quality software, but also the insider knowledge of the industry:

  • What skills will actually get you a job today?
  • What changes in your coding methodology can take you from an unrewarding job, to the best jobs out there?
  • What makes you an average developer lost in the crowd and what can make you stand out like an A player?
  • How you sell your expertise while still maintaining the upper hand in an interview?

We want to make the job market more approachable and predictable for the aspirants. We also want to help provide real value to the organisations hiring them, so that everybody wins. What about us? We are really passionate about teaming up with our fellow techies to get rid of the label of Indians developers being 'meagre'. We have got great results so far, and are super excited about the rest of the journey!

Front End Development
HTML/JavaScript/CSS Primer

HTML, JavaScript, CSS Primer

Course Description

We'll teach you the basics of the three core technologies required for building anything for the web. This is for people who have no prior experience with these technologies or never quite understood it in college.

This will enable you to build very simple web documents, but the major purpose of this course is to prepare you for the more advanced courses in Web Development.

In our knowledge, we are the only place providing a combined course for all of HTML, JS and CSS. We want to make the learning process affordable for you and more importantly, we understand the value of your time.
Frontend Development
Building Interactive Websites

Building Interactive Websites

Course Description

In this course you'll learn how to make awesome interactive website by combining the power of HTML, CSS and JavaScript. This will enable you to build websites for businesses such as restaurants, shops and colleges. A good example of such a website is the one you are viewing right now.

This course will also provide a smooth transition to the course on building web applications, which also interact with a database, and are much more complex in terms of functionality.

Frontend Development
Web Application Development

Web Application Development

Project Description

In this course we'll teach you how to build a modern web application. A web application is different from a website in the sense that it interacts with a backend database(DB) and generates web pages dynamically based on the data it recieves from the DB.

You'll learn the concepts of routing, DB integration, HTML templating and sessions among others. By the end of this course you'll have built a fully functional website like reddit/soundcloud.

Fronend Development
Web Application Development - Advanced

Project Title

Project Description

Advanced web application development will cover the concepts of Single Page Application, Client side HTML templating, API based development, client side routing and browser security issues among others.

Backend Development
Building APIs for Web Applications

Project Title

Project Description

APIs are very important in the context of web development. They let you decople your presentation layer with the database layer and also enable third parties to utilize the capabilities of your service.

We'll teach you how to build RESTful JSON APIs, the standards which should be followed while building them and how to ensure high performance with minimal coding.
